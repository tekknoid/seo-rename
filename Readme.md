# Seo Rename

Is a _simple_ massive media file renaming utility built in Rust.


### Features

- Accept dictionary of keywords.
- Check file existance to do not overwrite it.

### Usage

Shell (Mac, Linux):

```sh
./seo-rename /path/to/directory/ "keyword1,keyword2,keyword3"
```
